# TimeFlows 隐私政策

生效日期：【2026-02-21】

感谢你使用 TimeFlows（“本应用”）。我们重视你的隐私，并承诺保护你的个人信息。本隐私政策说明我们如何收集、使用、存储与共享你的信息。

## 1. 我们收集的信息
本应用在正常使用过程中可能会处理以下信息：

1) 你主动输入的数据  
- 时间记录内容（文本/语音转文本）  
- 标签、项目等自定义信息  
说明：这些内容默认保存在你的设备本地。

2) 设备与诊断信息（可选）  
- 应用崩溃日志、性能数据（若系统或你允许）  
说明：用于改进稳定性。

3) 第三方 AI 处理（如启用）  
- 当你使用 AI 功能时，输入内容可能会发送到你配置的 AI 服务提供商（如 OpenAI 或自定义接口）进行处理。  
- API Key 由你自行填写并保存在设备钥匙串（Keychain）中。

## 2. 我们如何使用信息
- 提供时间记录与统计功能  
- 支持 AI 分析与自动整理  
- 改善应用性能与稳定性

## 3. 信息存储与安全
- 数据默认保存在本地设备。  
- 若启用 iCloud 同步，数据会通过 Apple iCloud 同步到你的设备。  
- API Key 存储在 iOS Keychain 中。  

## 4. 信息共享
我们不会向第三方出售你的数据。  
仅在以下情况下可能共享：
- 你启用了 AI 功能，内容将发送到你指定的 AI 服务提供商  
- 法律要求或合规需要

## 5. 你的权利
- 你可以在应用内删除本地数据  
- 你可以停止使用 AI 功能或更换 AI 服务  
- 你可以关闭 iCloud 同步  

## 6. 未成年人
本应用不面向 13 岁以下用户。

## 7. 政策更新
如政策变更，我们会在本页面更新日期。
